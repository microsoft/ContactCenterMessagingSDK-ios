***Frequently Asked Questions***

1. How to integrate CCMSDK into iOS project?
   Refere https://github.com/microsoft/ContactCenterMessagingSDK-ios/blob/main/ContactCenterMessagingApp/README.md

2. Where do i found latest releases?
   You can get latest releases into 'Releases' section. https://github.com/microsoft/ContactCenterMessagingSDK-ios/releases

3. How do i manually download XCframeworks .zip file?
   You can go to Releases section. Go to your desired version. In assests you can get ContactCenterMessagingSDK.zip. Download .zip and now you can be able to use this XCframeworks.

4. Where do i found property customization details?
   You can refer this pdf for getting public properties which can you use for customization. https://github.com/microsoft/ContactCenterMessagingSDK-ios/blob/main/iOS_Widget_Customizations.pdf
